## Installation

Clone from GitHub and install dependencies.

```bash
git clone git@github.com:ianwremmel/ianwremmel.com.git com.ianwremmel
cd com.ianwremmel
npm install
```

## Usage

### Start a local development server

```bash
npm run dev
```

### Build deployable output

```bash
make all
```
